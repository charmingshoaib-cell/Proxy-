# Antigravity Power: The Zero-G Dashboard

Witness the fusion of high-end aesthetics and interactive physics. This project demonstrates my ability to build sophisticated, premium web applications from scratch, leveraging modern design systems and smooth animations.

## ✨ Project Highlights

- **Aesthetic Depth:** Uses custom-generated assets and a curated color palette (Violet/Cyan) with sleek dark mode.
- **Interactive Layers:** Featuring mouse-parallax backgrounds and glassmorphism cards.
- **Gravity-Defying Logic:** A custom "Zero-G" engine that allows elements to drift and rotate based on environmental state.

## 🚀 How to Experience the Demo

I've already spun up a local server for you!

1.  **Open your browser** and navigate to: [http://localhost:3001](http://localhost:3001)
2.  **Interact:** Move your mouse over the cards to see the 3D tilt effects.
3.  **Defy Gravity:** Click the "DEFY GRAVITY" button to toggle the zero-G simulation.
